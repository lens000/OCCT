# 计算点在直线上参数值的程序

这个程序用于计算3D空间中任意一点在给定直线上的参数值。

## 功能特性

- 支持3D空间中的点和直线
- 提供两种定义直线的方式：
  - 通过两个点
  - 通过一个点和方向向量
- 计算点在直线上的参数值t
- 计算点到直线的距离
- 计算直线上距离给定点最近的点
- 根据参数值计算直线上对应的点

## 数学原理

### 直线参数方程
直线L的参数方程为：P = P₀ + t × direction

其中：
- P₀ 是直线上的一点
- direction 是单位方向向量
- t 是参数值

### 参数值计算
对于任意点P，其在直线L上的参数值t通过以下公式计算：
t = (P - P₀) · direction

其中 · 表示点积运算。

### 点到直线距离
点P到直线L的距离为：
d = |P - P₀ - t × direction|

### 使用Makefile编译
```bash
cd src
make
```

```bash
./PointOnLineParam
```

```bash
make clean
```

### 使用CMake编译
```bash
mkdir build
cd build
cmake ../src
make
```

## 程序输出示例

程序会输出以下信息：
- 直线的定义（点和方向向量）
- 测试点的坐标
- 每个测试点在直线上的参数值t
- 直线上距离测试点最近的点
- 测试点到直线的距离

## 示例说明

### 示例1：通过两点定义直线
- 点P1: (0, 0, 0)
- 点P2: (1, 1, 1)
- 直线方向: (1/√3, 1/√3, 1/√3)

### 示例2：通过点和方向向量定义直线
- 点: (0, 0, 0)
- 方向: (1, 0, 0) - X轴方向

## 注意事项

- 程序使用双精度浮点数进行计算
- 方向向量会自动归一化为单位向量
- 参数值t可以是负数，表示点在直线的反向延长线上
- 当t=0时，点正好在直线上的参考点P₀处
